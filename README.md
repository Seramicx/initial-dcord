## 🏎️ Overview

**Initial Dcord** is a customized Discord theme powered by ClearVision v7, specifically tailored with an Initial D aesthetic. It features dynamic hover animations, a unique dark/translucent color palette, and hardware-accelerated UI tweaks to ensure maximum performance. 

## 🛠️ Installation

This theme is fully compatible with **BetterDiscord** and **Vencord**.

### Vencord Installation (Recommended)
1. Open your Discord Settings and navigate to the **Vencord** section -> **Themes**.
2. Put the `initial-dcord.theme.css` file inside your Vencord `themes` folder (usually located at `C:\Users\Username\AppData\Roaming\Vencord\themes`).
3. Turn on the theme inside the Vencord settings!

### BetterDiscord Installation
1. Open your Discord Settings and click on **Themes** under the BetterDiscord category.
2. Click the **Open Theme Folder** button.
3. Place `initial-dcord.theme.css` in that folder.
4. Enable the theme in your Discord settings.

## 📝 Features
- Fully integrated with the official ClearVision v7 layout.
- Custom Initial D background (AE86 artwork).
- GPU-accelerated micro-animations (slide-ins, scale-on-hover) that don't cause layout lag.
- Dark, clean, translucent chat cards fitting the midnight racing vibe.
- Specialized thread/spine alignment fixes.
